scTenifoldKnk: A Complete Tutorial
for Single-Cell Gene Virtual Knockout

Predict gene function using only wild-type single-cell data without real knockout experiments

Gene virtual knockout is a cutting-edge computational biology approach that predicts biological functions by simulating gene loss of function. It overcomes the high cost, long duration, and poor scalability of traditional experimental knockout methods such as CRISPR-Cas9. scTenifoldKnk is a machine learning tool designed specifically for this purpose, enabling efficient virtual knockout analysis using only wild-type single-cell transcriptomic data[1].

Its core principle is as follows: first, a robust, denoised wild-type single-cell gene regulatory network is constructed via random cell subsampling, principal component regression, and tensor decomposition. Next, a “pseudo-knockout” network is generated by zeroing out the entire row corresponding to the target gene in the adjacency matrix, simulating its functional loss. Finally, a manifold alignment algorithm maps the wild-type and pseudo-knockout networks into a unified low-dimensional space. By comparing Euclidean distances of gene projection positions, differentially regulated genes perturbed by the knockout are identified, and functional enrichment analysis is used to infer the target gene’s role[1].

Applications in Biomedical Research

The validity and utility of scTenifoldKnk have been demonstrated in multiple recent studies, supporting mechanistic dissection and therapeutic target discovery:

Diabetic Kidney Disease: Immune Microenvironment Targets

Xie et al. integrated single-cell RNA-seq, spatial transcriptomics, and Mendelian randomization to identify TNFRSF1A as a key causal mediator of immune microenvironment reprogramming in diabetic kidney disease. Subsequent virtual knockout revealed downstream regulation of apoptosis, VEGF, and NF‑κB pathways, suggesting novel therapeutic targets[2].

Environmental Toxicology: Air Pollution and Psoriasis

Xu et al. used systems toxicology to prioritize S100A9 and performed virtual knockout in psoriatic keratinocytes. Perturbation strongly affected epidermal differentiation and IL‑17 inflammatory signaling. Combined in vitro and in vivo validation supported a mechanism whereby air pollutants drive immune dysregulation via S100A9[3].

Colorectal Cancer: Immunotherapy Target Identification

Cui et al. combined transcriptome-wide Mendelian randomization and single-cell eQTL data during CD4+ T cell activation to nominate ORMDL3 and PARP14. Virtual knockout confirmed impacts on TNF‑α and NF‑κB signaling, providing candidate targets for cancer immunotherapy[4].

Neurodevelopment: Down Syndrome Mechanisms

Niu et al. built a fetal cortical cell atlas using snRNA-seq and performed virtual knockout. Dysregulated genes RUNX1 and APP on chromosome 21 were found to regulate tubulin family genes and STMN1/2, affecting neuronal development, axon regeneration, and dendritic transport, revealing molecular underpinnings of cortical abnormalities in Down syndrome[5].

Quick Start: Analysis Workflow

1. Data Preparation

scTenifoldKnk requires a raw UMI count matrix with genes as rows and cells as columns. Raw counts are strongly preferred over normalized data. For Seurat objects:

seurat_object <- readRDS("your_seurat_object.rds")
DefaultAssay(seurat_object) <- "RNA"
countMatrix <- GetAssayData(seurat_object, slot = "counts")

2. Run Virtual Knockout

Specify the target gene and launch the analysis:

result <- scTenifoldKnk(
    countMatrix = countMatrix,
    gKO = 'Casp4',
    nCores = 24,
    # Optional parameters below
    # qc = TRUE,
    # qc_mtThreshold = 0.1,
    # qc_minLSize = 1000,
    # nc_nNet = 10,
    # nc_nCells = 500,
    # nc_nComp = 3
)

3. Parameter Description

ParameterDefaultDescription
gKO–Target gene for virtual knockout
qc_mtThreshold0.1Max mitochondrial gene ratio
nc_nNet10Number of subsampled networks
nc_nCells500Cells per subnetwork
td_K3Tensor decomposition rank
ma_nDim2Manifold alignment dimension

4. Result Interpretation

The output contains three core components:

  • tensorNetworks: Denoised gene regulatory networks (wild-type & pseudo-knockout)
  • manifoldAlignment: Low-dimensional embeddings from manifold alignment
  • diffRegulation: Differentially regulated genes with p-values and logFC

Visualization

1. Volcano-like Bubble Plot

library(ggplot2)
library(dplyr)

df_KO <- result$diffRegulation
df_plot <- df_KO %>% mutate(neg_log10_pval = -log10(p.value))

p <- ggplot(df_plot, aes(x = logFC, y = neg_log10_pval)) +
    geom_point(aes(size = abs(logFC), color = logFC), alpha = 0.7) +
    scale_color_gradient2(low = "gray", mid = "gray", high = "red") +
    geom_hline(yintercept = -log10(0.05), linetype = "dashed") +
    geom_vline(xintercept = 0, linetype = "dashed") +
    geom_text(data = head(arrange(df_plot, p.value), 5), aes(label = gene), size = 3.5) +
    labs(x = "Log Fold Change", y = "-log10(P value)", title = "Differentially Regulated Genes") +
    theme_minimal()

print(p)

2. Top 10 Perturbed Genes Bar Plot

library(forcats)

df_top10 <- df_KO %>%
    arrange(desc(logFC)) %>%
    head(10) %>%
    mutate(neg_log10_pval = -log10(p.value),
           gene = fct_reorder(gene, logFC))

p <- ggplot(df_top10, aes(x = logFC, y = gene)) +
    geom_bar(aes(fill = neg_log10_pval), alpha = 0.8, width = 0.7) +
    geom_text(aes(label = sprintf("%.2f", logFC)), hjust = -0.2) +
    scale_fill_gradient(low = "lightblue", high = "darkblue") +
    scale_y_discrete(limits = rev) +
    labs(x = "Log Fold Change", y = "Gene", title = "Top 10 Affected Genes") +
    theme_minimal()

print(p)
