scTenifoldKnk Tutorial

Single-Cell Virtual Gene Knockout Analysis Framework

Introduction

scTenifoldKnk is a machine learning tool for single-cell virtual gene knockout analysis. It addresses the limitations of traditional gene knockout experiments, which are extremely costly and difficult to scale.

Key Advantages:
  • Only requires wild-type scRNA-seq data, no real knockout samples needed
  • Supports systematic virtual knockout of thousands of genes
  • Enables multi-gene simultaneous knockout simulation
  • Works with R, Python, and MATLAB implementations

Workflow Overview

The scTenifoldKnk framework consists of three core modules: Network Construction, Virtual Gene Knockout, and Manifold Alignment & Analysis.

Installation Guide

scTenifoldKnk is available in three different implementations:

R Version

# Install from GitHub
devtools::install_github('cailab-tamu/scTenifoldKnk')

# Load the package
library(scTenifoldKnk)

Python Version

# Install from PyPI
pip install scTenifoldpy

# Import the package
import scTenifoldpy as stk

MATLAB Version

The MATLAB version is available as part of the scGEAToolbox, which can be downloaded from GitHub.

Standard Analysis Pipeline

Here's a complete example of how to run scTenifoldKnk on your scRNA-seq data:

# Load your Seurat object
seurat_obj <- readRDS("your_seurat_object.rds")

# Extract raw count matrix
count_matrix <- GetAssayData(seurat_obj, assay = "RNA", layer = "counts")

# Run scTenifoldKnk with target gene 'Casp4'
result <- scTenifoldKnk(
    count_matrix,
    gKO = "Casp4",  # Target gene to knockout
    nc_nNet = 10,   # Number of subnetworks
    nc_nCells = 500, # Cells per subnetwork
    qc = TRUE,      # Enable quality control
    verbose = TRUE
)

Parameter Explanation

Parameter Default Description
X - Raw count matrix (genes × cells)
gKO - Target gene name to knockout
nc_nNet 10 Number of subnetworks to construct
nc_nCells 500 Number of cells to sample per subnetwork
qc TRUE Whether to perform quality control
qc_mtThreshold 0.1 Mitochondrial gene ratio threshold
qc_minLSize 1000 Minimum library size (UMIs)
td_K 3 Tensor decomposition rank
ma_nDim 2 Latent space dimension for manifold alignment

Runtime Reference

The runtime of scTenifoldKnk depends on the size of your dataset:

Dataset Size Estimated Runtime
300 cells / 1000 genes ~3.45 minutes
1000 cells / 2000 genes ~15 minutes
3000 cells / 3000 genes ~2 hours
7500 cells / 7500 genes ~10 hours
Tip: For large datasets, we recommend downsampling to ~3000 cells to speed up the analysis.

Result Interpretation

The result object contains three main components:

  • tensorNetworks: The denoised regulatory networks, including both wild-type and knockout versions
  • manifoldAlignment: The low-dimensional embeddings after manifold alignment
  • diffRegulation: The differential regulation results table, containing:
    • gene: Gene name
    • distance: Euclidean distance after projection
    • Z: Z-score
    • FC: Fold change
    • p.value: Raw p-value from chi-square test
    • p.adj: FDR adjusted p-value
# View the top differentially regulated genes
head(result$diffRegulation)

# Filter significant genes
sig_genes <- result$diffRegulation[result$diffRegulation$p.adj < 0.05, ]

Visualization

You can visualize the results using ggplot2:

1. Volcano Plot

library(ggplot2)

df <- result$diffRegulation
df$sig <- ifelse(df$p.adj < 0.05, "Significant", "Not Significant")

ggplot(df, aes(x = log2(FC), y = -log10(p.adj), color = sig)) +
    geom_point(alpha = 0.7) +
    geom_vline(xintercept = 0, linetype = "dashed", color = "gray") +
    geom_hline(yintercept = -log10(0.05), linetype = "dashed", color = "gray") +
    scale_color_manual(values = c("gray", "#e74c3c")) +
    theme_minimal() +
    labs(title = "Volcano Plot: Virtual Knockout of Casp4",
         x = "log2(Fold Change)", y = "-log10(Adjusted p-value)")

2. Top Differentially Regulated Genes

# Get top 10 genes
top10 <- head(df[order(-df$distance), ], 10)
top10$gene <- factor(top10$gene, levels = rev(top10$gene))

ggplot(top10, aes(x = distance, y = gene)) +
    geom_bar(stat = "identity", fill = "#3182ce", alpha = 0.8) +
    theme_minimal() +
    labs(title = "Top 10 Differentially Regulated Genes",
         x = "Manifold Distance", y = "Gene")

Published Applications

scTenifoldKnk has been successfully applied in multiple published studies:

  • Diabetic Nephropathy: Identified TNFRSF1A as a key regulator in kidney injury
  • Psoriasis: Revealed the pathogenic mechanism of air pollutants in skin inflammation
  • Colorectal Cancer: Discovered novel immunotherapy targets through systematic virtual screening
  • Down Syndrome: Uncovered the neurodevelopmental abnormalities mechanism in trisomy 21
